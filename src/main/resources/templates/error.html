Hilfe bei Validierungsfehlern:

  • Überprüfen Sie alle Pflichtfelder
  • Verwenden Sie das korrekte Datumsformat (DD.MM.YYYY)
  • Verwenden Sie das korrekte Zeitformat (HH:MM)

Ressource nicht gefunden:

  • Die angeforderte Gruppe oder der Student existiert möglicherweise nicht
  • Überprüfen Sie die URL auf Tippfehler
  • Kontaktieren Sie den Administrator, falls das Problem weiterhin besteht

Zugriff verweigert:

  • Sie haben möglicherweise nicht die erforderlichen Berechtigungen
  • Melden Sie sich mit einem anderen Benutzerkonto an
  • Kontaktieren Sie den Administrator für weitere Berechtigungen

Datenbankfehler:

  • Ein temporärer Datenbankfehler ist aufgetreten
  • Versuchen Sie es in ein paar Minuten erneut
  • Kontaktieren Sie den Administrator, falls das Problem anhält

Falls das Problem weiterhin besteht, kontaktieren Sie bitte den Systemadministrator.
